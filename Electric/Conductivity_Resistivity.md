## 电学里只有一条基本换算公式：

* $\rho =\dfrac{1}{\sigma}$
* $\sigma = electric conductivity（导电率，S · m⁻¹）$
* $\rho = electric resistivity（电阻率，Ω · m）$

只要有某温度 ***T*** 下的 $\sigma(T)$，直接取倒数就是 $\rho(T)$。 
下面重点讨论软件（例如 STAR-CCM⁺、Fluent、COMSOL）经常要你输入的三组参数—— 
 
1. ρ₀   （reference resistivity，Ω · m） 
2. T₀   （reference temperature，K 或 °C） 
3. α   （temperature coefficient of resistivity，K⁻¹） 
 
并说明如何从“不同温度下的测得/给定导电率”反推它们。
 
----------------------------------------------------------------
一、温度依赖的一阶近似
----------------------------------------------------------------
工业软件默认都用线性模型（有效范围通常在 -50 ~ +200 °C）：

  $\rho(T) = \rho_0[1 + \alpha(T - T_0)] $

  $\sigma(T) = \dfrac{1}{\rho(T)} =\dfrac{1}{\rho_0[1+\alpha(T-T_0)]} $
  
  (也有人把导电率写成 $\sigma(T)=\sigma_0[1 - \alpha(T-T_0)] $, 其中 $\sigma_0 = 1/\rho_0$，符号正负相反但数值同一。）
  
  只要手里有至少两组 (T, σ) 数据，就能解出 **ρ₀、α**； 有 3 组及以上数据时，用线性回归更稳健。

----------------------------------------------------------------
二、两点法（最简手算）
----------------------------------------------------------------
已知 (T₁, σ₁)、(T₂, σ₂)，先转成 ρ：

 
 $$\rho_1 = \dfrac{1}{\sigma_1}, \qquad
          \rho_2 = \dfrac{1}{\sigma_2}$$

 
再选定参考温度 $T_0$（通常 293.15 K 或 20 °C，也可取 $T_1$）：
 
$$\rho_0 = \rho_1 / [1 + \alpha(T_1 - T_0)]$$
 
而 
  $$\alpha = \dfrac{\rho_2 - \rho_1} {\rho_1(T_2 - T_1)}$$
 
步骤汇总（选 $T_0 = T_1$ 最方便） 
1. $\rho_0 = \rho_1$
2. $\alpha = (\rho_2 - \rho_1)/( \rho_1 (T_2 - T_1))$


  举例:
  
    σ₁ = 5.96 × 10⁷ S/m @ 20 °C 
    σ₂ = 5.88 × 10⁷ S/m @ 80 °C
    ρ₁ = 1/σ₁ ≈ 1.677 × 10⁻⁸ Ω·m
    ρ₂ = 1/σ₂ ≈ 1.700 × 10⁻⁸ Ω·m 
    α ≈ (1.700-1.677)/(1.677×60) ≈ 2.3 × 10⁻³ K⁻¹
  最终参数
  
    ρ₀ = 1.677 × 10⁻⁸ Ω·m @ T₀ = 20 °C
    α = 2.3 × 10⁻³ K⁻¹

----------------------------------------------------------------
三、多点法（推荐，用 Excel/Python 回归）
----------------------------------------------------------------
如果有 N ≥ 3 组数据，先把所有 σ → ρ，接着拟合： 

  1) 设 $y_i = \dfrac{\rho_i}{\rho_0} - 1$
  2) 对 $y_i$ 与 $x_i = T_i - T_0$ 做线性回归，斜率即 α 
  
  更通俗：直接对 $\rho_i = a + b\,T_i$ 做回归，结果
  
    ρ₀ = a + b T₀ 
    α = b/ρ₀
 
Python 样例（pandas + numpy）：
 
```python
import numpy as np, pandas as pd
# csv 两列: T(°C), sigma(S/m)
df = pd.read_csv('sigma_vs_T.csv')
df['rho'] = 1 / df['sigma']
 
T0 = 20.0                 # 参考温度
X  = df['T'] - T0
Y  = df['rho']
 
# 一元线性回归
b, a = np.polyfit(X, Y, 1)   # Y = a + b*X
rho0 = a                     # 因为 X=0 时就是 T=T0
alpha = b / rho0
 
print(f"rho0 = {rho0:.4e} Ω·m @ {T0} °C")
print(f"alpha = {alpha:.4e} 1/K")
```
 
把得到的 ρ₀、α、T₀ 直接录进软件即可。

----------------------------------------------------------------
四、注意事项
----------------------------------------------------------------
1. 线性假设只对多数金属在室温附近成立；高温或半导体材料需用多项式或表格插值。 
2. 电导率/电阻率温度系数在文献上可能给 **conductivity 系数 β**： 
  $$\sigma(T)=\sigma_0[1+\beta(T-T_0)\,]$$ 
  则 $$\beta = -\alpha$$
  输入前看清定义。 
3. 单位一致：软件若要求 **°C**，α 应写成 “每 °C”；要求 **K** 就用 “每 K”——数值相同。 
4. STAR-CCM⁺ 等可直接接受 “表格” 材料属性；若温区跨度大、不想线化，可在 Material → Electric Conductivity → Method 选 “Table” 逐点填值，省去拟合。

----------------------------------------------------------------
五、一句话结论
----------------------------------------------------------------
* 导电率 ↔ 电阻率：互为倒数。
* 有温-σ 数据 → 取倒数得 ρ → 线性（或回归）求出 ρ₀、α、T₀。 
* ρ₀ 取某参考温度对应值即可；α = 相对电阻率变化率。这样就能把 “散点导电率” 转成软件常用的三参数电阻率模型。
